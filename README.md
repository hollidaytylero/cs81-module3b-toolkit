# cs81-module3b-toolkit
CS81 Math Toolkit Functions
Purpose
This assignment helps you explore JavaScript functions by building your own custom math toolkit. You’ll start with a simple function and gradually build a more complex, reusable set of functions. You will document your process, track your predictions, and reflect on how the toolkit evolves over time.

Learning Objectives
Create, test, and improve JavaScript functions
Use Git and GitHub to document development over time
Understand and apply functional thinking
Practice debugging and writing clean, reusable code
Reflect on patterns, logic, and unexpected outcomes
Repository Setup
GitHub Repo Name: cs81-module3b-toolkit

Public or private is fine
Add a description like “CS81 Math Toolkit Functions”
Include a README file
Project Instructions
Create a file called math.js inside your GitHub repo.
Start by writing a simple function, like double(n).
Commit and push your function with a clear message (e.g., “Add double function”).
Write a short prediction of what the function will do in a file called PREDICTIONS.md.
Test the function and record the output in RESULTS.md.
Repeat steps 2–5 at least 5 times with new or modified functions:
Try square(), isEven(), isOdd(), multiply(), etc.
Make each new commit a meaningful step
After all functions are complete, create a REFLECTION.md file:
What patterns or surprises did you notice?
What logic challenged your thinking?
How might this kind of toolkit be used in the real world?
Submission
1. Submit the public GitHub link to your repo named cs81-module3b-toolkit. Make sure your repository contains:

math.js – your math functions
PREDICTIONS.md – your expected results
RESULTS.md – your tested results
REFLECTION.md – your final thoughts
Minimum of 5 separate commits
